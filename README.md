# leveluptuts:svelte-accounts-ui

## Basic

```js
<script>
import { LoginWindow } from 'leveluptutorials:svelte-accounts-ui'
</script>

<LoginWindow />
```

## You can also

```js
<script>
import { Login } from 'leveluptutorials:svelte-accounts-ui'
</script>

<Login />
```

```js
<script>
import { Signup } from 'leveluptutorials:svelte-accounts-ui'
</script>

<Signup />
```
